# Turbo Happy Ghast

The Happy Ghast's default speed leaves a lot to be desired. This Mod and Data Pack significantly improves its flight speed the moment you mount it, making it much faster.

## How it works

Mojang added the `minecraft:flying_speed` attribute to Happy Ghasts in snapshot
[25w20a](https://www.minecraft.net/en-us/article/minecraft-snapshot-25w20a) and this solution takes full advantage of it. While you are mounted on a Happy Ghast, the attribute is bumped up to make it noticeably faster. The moment you dismount, it goes right back to its default value.
