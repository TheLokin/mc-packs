# Cheaper Shulker Boxes

Cheaper Shulker Boxes is a vanilla friendly Mod and Data Pack that fixes inventory clutter by making storage much more accessible.

## Features

- **Half the cost:** The recipe now requires only **1 Shulker Shell and 1 Chest** instead of two shells. Every single shell you loot now equals a full Shulker Box.
- **Crafting on the go:** Since the recipe only uses two ingredients, it fits perfectly inside the 2x2 crafting grid of your inventory. **No crafting table required!**
- **Vanilla like:** No custom items or complex mechanics. Just a simple vanilla tweak.
